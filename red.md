# Antigravity

> A paid, execution-locked productivity system that eliminates choice, excuses, and drift.

Antigravity is **not a habit tracker** and **not a motivational app**.  
It is a **time-boxed execution contract** where users *define their own obligations*, temporarily surrender flexibility, and are penalized for non-compliance.

This system is designed for:
- Competitive exam aspirants
- Founders and builders
- High-agency individuals who want results, not reminders

## ğŸ”’ How It Works (System Flow)

### 1. User-Defined Obligation
- User manually types:
  - Syllabus
  - Tasks
  - Study blocks
  - Duration (7 / 14 / 30 days)
- No templates by default
- Ownership is enforced at input level

---

### 2. Execution Lock
- Once started:
  - Tasks cannot be edited
  - Schedule cannot be softened
  - Scope cannot be reduced
- The system treats the plan as **immutable**

---

### 3. Daily Verification
- Completion is verified via:
  - Manual proof (check + friction)
  - Optional evidence (notes, screenshots, logs)
- Missed tasks are recorded, not forgiven

---

### 4. Failure Threshold
- After **N failures** (configurable):
  - App access is locked
  - No planning, no tracking, no edits
- User is shown a failure summary

---

### 5. Paid Re-Entry
- To regain access:
  - User must explicitly re-enter
  - Payment is required
  - Previous data remains read-only
- This creates **loss aversion and seriousness**
